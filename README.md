YKD-RF Bluetooth LED Integration for Home Assistant
Ця користувацька інтеграція дозволяє підключити бюджетні Bluetooth LED-стрічки (моделі YKD-RF та аналоги) безпосередньо до Home Assistant без сторонніх хабів чи хмарних сервісів.

✨ Основні можливості
Миттєвий відгук (Fast Response): Технологія Keep-Alive утримує Bluetooth-сесію відкритою протягом 60 секунд після останньої команди. Це прибирає затримки на "рукостискання" при повторному керуванні.

Керування яскравістю: Плавне регулювання від 1% до 100% за допомогою прямих HEX-команд.

Режими ефектів: Підтримка анімацій (Strobe, Fade, Jump, Slow Flash) з можливістю миттєвого повернення до статичного світла через пункт "None".

Енергоефективність: Автоматичне розірвання з'єднання після хвилини бездіяльності для звільнення Bluetooth-адаптера сервера.

Повністю локально: Працює без інтернету через бібліотеку Bleak.

🛠 Технічні деталі
Протокол: BLE (Bluetooth Low Energy).

UUID Характеристики: 0000ffd9-0000-1000-8000-00805f9b34fb.

Сумісність: Тестовано на контролерах з MAC-префіксом 02:05:11.

🚀 Встановлення
Спосіб 1: Через HACS (Рекомендовано)
Це найшвидший спосіб, який дозволяє отримувати оновлення прямо в інтерфейсі HA.

Відкрийте HACS у вашому Home Assistant.

Натисніть на три крапки (⋮) у верхньому правому куті та оберіть Custom repositories (Користувацькі репозиторії).

У полі Repository вставте посилання: https://github.com/callmeloony/YKD-RF-Bluetooth-LED

У полі Category оберіть Integration.

Натисніть Add, а потім знайдіть та встановіть YKD-RF Bluetooth LED.

Перезавантажте Home Assistant.

Спосіб 2: Вручну
Завантажте репозиторій та скопіюйте папку ykd_led у директорію вашого сервера /config/custom_components/.

Перезавантажте Home Assistant.

⚙️ Налаштування
Перейдіть у Налаштування -> Пристрої та служби.

Натисніть Додати інтеграцію та введіть у пошуку YKD-RF Bluetooth LED.

Вкажіть довільну назву пристрою та його MAC-адресу (наприклад, 02:05:11:00:03:EB).

Насолоджуйтесь миттєвим керуванням!
